# ⚽ C++ Football Match Simulator

A **console-based football match simulator** built in **C++** that allows users to select leagues and teams, adjust difficulty levels, and simulate a complete football match with randomized gameplay events and real-time commentary.

 Features
- League selection system  
- Team selection from chosen league  
- Random opponent selection  
- Difficulty levels: Easy, Normal, Hard  
- Real-time match simulation with:
  - Goals  
  - Saves  
  - Missed shots  
  - Offside events  
  - Half-time and full-time commentary  
- Man of the Match selection

##  Concepts & Learning

This project demonstrates fundamental C++ concepts:

- Structs for team and league modeling  
- Vectors (`std::vector`) for dynamic data storage  
- Functions & modular programming  
- Random number generation (`rand()`)  
- Loops, conditionals, and control flow  
- User input handling via console  
- Basic game simulation logic
